# 🚗 자동차 디지털 키 보안 시스템 프로토타입

## 📝 프로젝트 설명
- 자동차의 기초 보안 개념인 '인증(Authentication)'을 실습하기 위한 프로토타입입니다.

## ✅ 현재 구현된 기능
- **PIN 인증**: 지정된 4자리 번호(1234) 입력 시 잠금 해제.
- **보안 로직**: 3회 이상 인증 실패 시 시스템 잠금 기능 작동.
